## Merge Branches

Merge `<branch-x>` into the current branch.
```shell
git merge <branch-x>
```

## Delete Branch

Delete `<branch-x>`.
```shell
git branch -d <branch-x>
```
??? Options "Options _git branch_"

    `-dr <remote/branch-name>` delete a remote branche
